== Description ==
プロフィールのページを作成することができます。プラグイン内で作成したソースを投稿や固定ページに投稿して使用します。
プロフィールページ内にはJavaScriptが組み込まれているので動きのあるページがご堪能いただけます。

== Installation ==
プラグインページの新規追加から追加してください。
